#### Стакан:

##### Основная информация:

| Видим/ищем         | Что делать?       |
| ------------------ | ----------------- |
| Ликвидность        | Настройки стакана |
| Роботов            | Объёмы >500k$     |
| Спуферов           | Сжатие            |
| Крупных участников |                   |


1. Круглые числа усиливают плотность
2. Трейдер должен **использовать плотность** либо в **отскок**, либо в **проедание**. *Плотность - одно из оснований*
3. Работа от **спуфера** приводит к **убыткам**

##### Настоящая плотность:

1. Стоит заранее
2. Стоит на круглом числе
3. Стоит на уровне
4. Стоит на споте
5. Плотность не снимается при подходе цены к ней
6. Плотность не стоит для того, чтобы вы заработали

##### Пример *робота*:

***Рулет*** - алгоритм, который вкручивает объём по определённой цене, удерживает его, через какое-то время снимается, переставляя объём, при этом влияя своими действиями на цену.

#### Лента:

##### Основная информация:
1. Скорость прохождения принтов
2. Ускорение прохождения принтов
3. Размер заявок (растёт или падает)
4. Закономерность (робот)
