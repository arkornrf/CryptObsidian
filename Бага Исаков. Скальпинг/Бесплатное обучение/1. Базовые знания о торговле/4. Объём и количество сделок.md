[[3. Элементы торгового ПО]]

#### Стадии адекватного движения:

**3 вида зон актива**
- Red - боковик (*плохо*, нельзя торговать)
- Green - активный рост актива (*отличный* период для торговли)
- Blue - (*хорош* для торговли) зона определяется как актив, который после сильного движения цены сохраняет с поводырём (BTC) и ещё не ушёл в зону до роста активности (т.е. Red)

*Актив в игре считается только в зелёной и синей зонах*

#### Механика рынка:

- Мы зарабатываем там, где другие теряют, либо вынужденно раздают деньги
- Рынок не генерирует деньги, он их распределяет
- Объём + график + уровень + лента + плотность = правильный скальпинг

#### Основания для сделки:

|  Лучшие   |    Средние     |  Не являются ими   |
| :-------: | :------------: | :----------------: |
|  Уровень  | Скорость ленты | Короткий stop-loss |
|  Паттерн  |     Робот      |     Потенциал      |
| Плотность |    Поводырь    |    Воображение     |
|   Объём   |                |   Чужое решение    |
